# House Price Prediction Model
A machine learning project to predict house prices using multiple regression algorithms. This project demonstrates end-to-end implementation 
including data preprocessing, data analysis, feature engineering, and model comparison.
## Project Overview
This project predicts house sale prices using different machine learning models:
- **Support Vector Regression (SVR)**
- **Random Forest Regressor**
- **Linear Regression**
The models are wvaluated and compared using Mean Absolute Percentage Error to determine the most accurate approach.
## Technologies
- **Pandas** - Data manipulation and preprocessing
- **Scikit-learn** - Machine learning models and preprocessing
- **Matplotlib/Seaborn** - Data visualization and EDA
- **OpenPyXL** - Excel file handling
## Key Features
- **Data Preprocessing**: Handles missing values, identifies categorical/numerical features
- **Exploratory Data Analysis**: Visualizes correlations and feature distribuitions
- **Feature Engineering**: One-Hot encoding for categorical variables, feature scaling
- **Model Evaluation**: Uses Mean Absolute Percentage Error for performance comparison
